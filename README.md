# Computer Vision 2026

Landing page for the USI Master-level course project ideas gallery.

Students can propose a project idea by opening a pull request that edits `index.html`.
GitHub organization:
https://github.com/Computer-Vision-2026

Copy the example `<article class="project-card">` block in the project section and add:

- a teaser image or visual
- a short topic line
- a project title
- a 60-120 word pitch

Course information is based on the official USI profile:
https://search.usi.ch/courses/35275471/computer-vision-pattern-recognition
